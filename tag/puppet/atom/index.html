But if instead you do this:</p> <div class="highlight"><pre><span class="n">class</span> <span class="n">bar</span> <span class="n">inherits</span> <span class="n">foo</span> <span class="p">{</span> <span class="n">file</span> <span class="p">{</span> <span class="err">&#39;</span><span class="o">/</span><span class="n">tmp</span><span class="o">/</span><span class="n">myfile</span><span class="err">&#39;</span><span class="o">:</span> <span class="p">}</span> <span class="p">}</span> </pre></div> <p>It will be created with mode 0600. In other words, if you use inherits then definitions in the parent class are available in the scope of your subclass. If you include, then definitions in he included class are "below" the scope of the including class.</p>
