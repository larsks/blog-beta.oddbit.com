Odd Bitshttp://blog-beta.oddbit.com/One of these things is not like the othersSun, 17 Nov 2013 00:00:00 -0500json-tools: cli for generating and filtering jsonhttp://blog-beta.oddbit.com/2013/11/17/json-tools/<p>Interacting with JSON-based APIs from the command line can be difficult, and OpenStack is filled with REST APIs that consume or produce JSON. I've just put pair of tools for generating and filtering JSON on the command line, called collectively <a href="http://github.com/larsks/json-tools/">json-tools</a>.</p> <p>Both make use of the Python <a href="https://github.com/akesterson/dpath-python">dpath</a> module to populate or filter JSON objects.</p> <p>The <code>jsong</code> command generates JSON on <code>stdout</code>. You provide <code>/</code>-delimited paths on the command line to represent the JSON structure. For example, if you run:</p> <div class="highlight"><pre><span class="err">$</span> <span class="n">jsong</span> <span class="n">auth</span><span class="o">/</span><span class="n">passwordCredentials</span><span class="o">/</span><span class="n">username</span><span class="o">=</span><span class="n">admin</span> \ <span class="n">auth</span><span class="o">/</span><span class="n">passwordCredentials</span><span class="o">/</span><span class="n">password</span><span class="o">=</span><span class="n">secret</span> </pre></div> <p>You get:</p> <div class="highlight"><pre><span class="p">{</span> <span class="s">&quot;auth&quot;</span><span class="o">:</span> <span class="p">{</span> <span class="s">&quot;passwordCredentials&quot;</span><span class="o">:</span> <span class="p">{</span> <span class="s">&quot;username&quot;</span><span class="o">:</span> <span class="s">&quot;admin&quot;</span><span class="p">,</span> <span class="s">&quot;password&quot;</span><span class="o">:</span> <span class="s">&quot;secret&quot;</span> <span class="p">}</span> <span class="p">}</span> <span class="p">}</span> </pre></div> <p>The <code>jsonx</code> command accepts JSON on <code>stdin</code> and selects subtrees or values for output on <code>stdout</code>. Given the above output, you could extract the password with:</p> <div class="highlight"><pre><span class="n">jsonx</span> <span class="o">-</span><span class="n">v</span> <span class="n">auth</span><span class="o">/</span><span class="n">passwordCredentials</span><span class="o">/</span><span class="n">password</span> </pre></div> <p>Which would give you:</p> <div class="highlight"><pre><span class="n">secret</span> </pre></div> <p>The <code>-v</code> flag here indicates that you only want the values of matched paths; without the <code>-v</code> the output would have been:</p> <div class="highlight"><pre><span class="n">auth</span><span class="o">/</span><span class="n">passwordCredentials</span><span class="o">/</span><span class="n">password</span> <span class="n">secret</span> </pre></div> <p>There are more examples -- including some use of the OpenStack APIs -- in the <a href="https://github.com/larsks/json-tools/blob/master/README.md">README</a> document.</p>Lars Kellogg-StedmanSun, 17 Nov 2013 00:00:00 -0500tag:blog-beta.oddbit.com,2013-11-17:2013/11/17/json-tools/jsonopenstack
