Odd Bitshttp://blog-beta.oddbit.com/2010-01-29T00:00:00-05:00Retrieving Blogger posts by post id2010-01-29T00:00:00-05:00Lars Kellogg-Stedmantag:blog-beta.oddbit.com,2010-01-29:2010/01/29/retrieving-blogger-posts-by-post-id/<p>I spent some time recently trying to figure out, using Google's <a href="http://code.google.com/apis/gdata/docs/2.0/basics.html">gdata</a> API, how to retrieve a post from a <a href="http://www.blogger.com/">Blogger</a> blog if I know corresponding post id. As far as I can tell there is no obvious way of doing this, at least not using the gdata.blogger.client api, but after much nashing of teeth I came up with the following solution.</p> <p>Given client, a <a href="http://gdata-python-client.googlecode.com/svn/trunk/pydocs/gdata.blogger.client.html">gdata.blogger.client</a> instance, and blog, a <a href="http://gdata-python-client.googlecode.com/svn/trunk/pydocs/gdata.blogger.data.html">gdata.blogger.data.Blog</a> instance, the following code will return a <a href="http://gdata-python-client.googlecode.com/svn/trunk/pydocs/gdata.blogger.data.html">gdata.blogger.data.BlogPost</a> instance:</p> <div class="highlight"><pre><span class="n">post</span> <span class="o">=</span> <span class="n">client</span><span class="p">.</span><span class="n">get_feed</span><span class="p">(</span><span class="n">blog</span><span class="p">.</span><span class="n">get_post_link</span><span class="p">().</span><span class="n">href</span> <span class="o">+</span> <span class="err">&#39;</span><span class="o">/%</span><span class="n">s</span><span class="err">&#39;</span> <span class="o">%</span> <span class="n">post_id</span><span class="p">,</span> <span class="n">auth_token</span><span class="o">=</span><span class="n">client</span><span class="p">.</span><span class="n">auth_token</span><span class="p">,</span> <span class="n">desired_class</span><span class="o">=</span><span class="n">gdata</span><span class="p">.</span><span class="n">blogger</span><span class="p">.</span><span class="n">data</span><span class="p">.</span><span class="n">BlogPost</span><span class="p">)</span> </pre></div> <p>I'm not sure if this is the canonical solution or not, but it appears to work for me.</p>
