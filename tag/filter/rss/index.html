Odd Bitshttp://blog-beta.oddbit.com/One of these things is not like the othersWed, 10 Feb 2010 00:00:00 -0500Filtering Blogger feedshttp://blog-beta.oddbit.com/posts/2010/02/10/filtering-blogger-feeds/<p>After encountering a number of problems trying to filter Blogger feeds by tag (using services like <a href="http://feedrinse.com/">Feedrinse</a> and Yahoo <a href="http://pipes.yahoo.com/">Pipes</a>), I've finally put together a solution that works:</p> <ul> <li>Shadow the feed with <a href="http://feedburner.com/">Feedburner</a>.</li> <li>Enable the <em>Convert Format Burner</em>, and convert your feed to RSS 2.0.</li> <li>Use Yahoo <a href="http://pipes.yahoo.com/">Pipes</a> to filter the feed (because Feedrinse seems to be broken).</li> </ul> <p>This let me create a feed that excluded all my posts containing the <em>fbpost</em> tag, thus allowing me to avoid yet another postgasm in Facebook when adding new import URL to notes.</p> <p>While fiddling with this I came across <a href="http://www.tothepc.com/archives/10-tools-to-combine-mix-blend-multiple-rss-feeds/">this article</a> that discusses a number of tools (some no longer available) for processing RSS feeds.</p>Lars Kellogg-StedmanWed, 10 Feb 2010 00:00:00 -0500tag:blog-beta.oddbit.com,2010-02-10:posts/2010/02/10/filtering-blogger-feeds/pipesrssatombloggerfilter
