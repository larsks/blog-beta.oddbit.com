PowerShellhttp://blog-beta.oddbit.com/posts/2012/11/04/powershell-random-passwords/<p>I was looking for PowerShell solutions for generating a random password (in order to set the Administrator password on a Windows instance provisioned in <a href="http://www.openstack.org/">OpenStack</a>), and found several solutions using the GeneratePassword method of <code>System.Web.Security.Membership</code> (documentation <a href="http://msdn.microsoft.com/en-us/library/system.web.security.membership.generatepassword.aspx">here</a>), along the lines of <a href="https://gist.github.com/4011878">this</a>:</p> <div class="highlight"><pre><span class="n">Function</span> <span class="n">New</span><span class="o">-</span><span class="n">RandomComplexPassword</span> <span class="p">(</span><span class="err">$</span><span class="n">length</span><span class="o">=</span><span class="mi">8</span><span class="p">)</span> <span class="p">{</span> <span class="err">$</span><span class="n">Assembly</span> <span class="o">=</span> <span class="n">Add</span><span class="o">-</span><span class="n">Type</span> <span class="o">-</span><span class="n">AssemblyName</span> <span class="n">System</span><span class="p">.</span><span class="n">Web</span> <span class="err">$</span><span class="n">password</span> <span class="o">=</span> <span class="p">[</span><span class="n">System</span><span class="p">.</span><span class="n">Web</span><span class="p">.</span><span class="n">Security</span><span class="p">.</span><span class="n">Membership</span><span class="p">]</span><span class="o">::</span><span class="n">GeneratePassword</span><span class="p">(</span><span class="err">$</span><span class="n">length</span><span class="p">,</span><span class="mi">2</span><span class="p">)</span> <span class="k">return</span> <span class="err">$</span><span class="n">password</span> <span class="p">}</span> </pre></div> <p>While this works, I was unhappy with the generated passwords: they were difficult to type or transcribe because they make heavy use of punctuation. For example:</p> <ul> <li><code>(O;RK_wx(IcD;&lt;V</code></li> <li><code>+3N)lkU5r)nHiL#</code></li> </ul> <p>These looks more like line noise (remember that? No? Get off my lawn...) than anything else and feel very unnatural to type.</p> <p>I was looking for longer strings consisting primarily of letters and digits. Thanks to Hey, Scripting Guy I learned about the Get-Random and ForEach-Object methods (and the % alias for the latter), and ended up with <a href="https://gist.github.com/4011916">the following</a>:</p> <div class="highlight"><pre><span class="err">#</span> <span class="nx">Generate</span> <span class="nx">a</span> <span class="nx">random</span> <span class="nx">password</span> <span class="err">#</span> <span class="nb">Usage</span><span class="p">:</span> <span class="nx">random</span><span class="na">-password</span> <span class="o">&lt;</span><span class="nb">length</span><span class="o">&gt;</span> <span class="nx">Function</span> <span class="nx">random</span><span class="na">-password</span> <span class="p">(</span><span class="nv">$length</span> <span class="o">=</span> <span class="mi">15</span><span class="p">)</span> <span class="p">{</span> <span class="nv">$punc</span> <span class="o">=</span> <span class="mi">46</span><span class="nx">..46</span> <span class="nv">$digits</span> <span class="o">=</span> <span class="mi">48</span><span class="nx">..57</span> <span class="nv">$letters</span> <span class="o">=</span> <span class="mi">65</span><span class="nx">..90</span> <span class="o">+</span> <span class="mi">97</span><span class="nx">..122</span> <span class="err">#</span> <span <span class="na">-count</span> <span class="nv">$length</span> <span class="sb">`</span> <span class="sb"> -input ($punc + $digits + $letters) |</span> <span class="sb"> % -begin { $aa = $null } `</span> <span class="na">-process</span> <span class="p">{</span><span class="nv">$aa</span> <span class="o">+=</span> <span class="err">[</span><span class="nx">char</span><span class="cp">]</span>$_} ` -end {$aa} return $password } </pre></div> <p>This generates strings of letters and digits (and ".") that look something like:</p> <ul> <li><code>2JQ0bW7VMqcm4UB</code></li>
