Odd Bitshttp://blog-beta.oddbit.com/2010-02-07T00:00:00-05:00MBTA realtime XML feed2010-02-07T00:00:00-05:00Lars Kellogg-Stedmantag:blog-beta.oddbit.com,2010-02-07:2010/02/07/mbta-realtime-xml-feed/<p>The <a href="http://mbta.com/">MBTA</a> has a trial web service interface that provides access to realtime location information for select MBTA buses, as well as access to route information, arrival prediction, and other features. More information can be found here:</p> <blockquote> <p><a href="http://www.eot.state.ma.us/developers/realtime/">http://www.eot.state.ma.us/developers/realtime/</a></p> </blockquote> <p>The service is provided by <a href="http://www.nextbus.com/">NextBus</a>, which specializes in real-time location information for public transit organizations. The <a href="http://www.eot.state.ma.us/developers/downloads/MBTA_XML_Feed_Trial_Docs_13Nov09.pdf">API</a> (sorry, PDF) is very simple and does not require any sort of advance registration.</p> <p>At the moment, the service only provides coverage for a small number of routes (39, 111, 114, 116, 117). I hope they expand the coverage of this service in the near future!</p>
