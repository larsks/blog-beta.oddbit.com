It supports both read-only and read-write sessions, and unlike many other tools it does not require any software installation on the remote side. This makes it tremendously handy for:</p> <ul> <li>Streaming terminal demonstrations to a diverse audience, or</li> <li>Sharing a terminal session with someone without needing to much about with ssh, tmux, screen, etc.</li> </ul> <p>I've successfully used <a href="https://github.com/progrium/termshare">Termshare</a> under both Fedora (19 and 20) and CentOS. To get started on these platforms, you'll need to install the <a href="http://golang.org/">Go</a> language, <a href="http://git-scm.org/">git</a> for cloning the termshare repository, and <a href="http://mercurial.selenic.com/">mercurial</a> to support installation of some Go libraries:</p> <div class="highlight"><pre><span class="c"># yum -y install golang git hg</span> </pre></div> <p>Then, clone the <a href="https://github.com/progrium/termshare">Termshare</a> repository:</p> <div class="highlight"><pre><span class="err">$</span> <span class="n">git</span> <span class="n">clone</span> <span class="n">https</span><span class="o">:</span><span class="c1">//github.com/progrium/termshare.git</span> <span class="err">$</span> <span class="n">cd</span> <span class="n">termshare</span> </pre></div> <p>Install the Go dependencies:</p> <div class="highlight"><pre><span class="err">$</span> <span class="n">mkdir</span> <span class="n">gopath</span> <span class="err">$</span> <span class="n">export</span> <span class="n">GOPATH</span><span class="o">=</span><span class="err">$</span><span class="n">PWD</span><span class="o">/</span><span class="n">gopath</span> <span class="err">$</span> <span class="n">go</span> <span class="n">get</span> </pre></div> <p>This will yield...</p> <div class="highlight"><pre><span class="n">go</span> <span class="n">install</span><span class="o">:</span> <span class="n">no</span> <span class="n">install</span> <span class="n">location</span> <span class="k">for</span> <span class="n">directory</span> <span class="o">/</span><span class="n">home</span><span class="o">/</span><span class="n">lars</span><span class="o">/</span><span class="n">src</span><span class="o">/</span><span class="n">termshare</span> <span class="n">outside</span> <span class="n">GOPATH</span> </pre></div> <p>...but you can ignore that.
