populates a number of variables that can be used in your scripts:</p> <ul> <li><code>XMLSTARTELEM</code> marks the start of a new element (and is set to the name of that element).</li> <li><code>XMLATTR</code> is available when <code>XMLSTARTELEM</code> is set and contains the element attributes.</li> <li><code>XMLENDELEM</code> marks the end of an element (and is set to the name of the element).</li> </ul> <p>There are other variables available, but with this basic set is becomes easy to extract information from XML documents.</p>
